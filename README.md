# Aura Chique - API Central

Serviço de backend do e-commerce Aura Chique, desenvolvido em Node.js com o framework Express. 

Esta API é responsável por processar todas as regras de negócio da aplicação. Ela fornece endpoints estruturados para a gestão de usuários, operações de CRUD do catálogo (produtos, categorias e variações de grade) e processamento do fluxo de checkout. A segurança é mantida através de autenticação via JWT, e a persistência de dados ocorre em um banco relacional MySQL.
